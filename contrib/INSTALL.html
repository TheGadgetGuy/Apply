Description

It makes a post in a forum specified in the ACP with the answers to the questions that can be added in the ACP. Supports Textbox, Inputbox, Radiobuttons, Selectbox, Checkboxes. uses the wowchar bbCode
Posts can either be public or private. this can be set in the ACP.

Requirements

  1. bbdkp 1.1.0 or higher
  2. bbTips 0.3.5 or higher
  3. phpbb3.0.7-PL1 or higher
  4. ftp and founder admin access on your phpbb installation.

Installation

Quick install guide

  1. Preparation : make two forums, one for Silent applications, one for public applications.

  2. Press Install to let Automod copy the files and do the core edits.
  3. Browse to install/installapply.php and run and the apply plugin table installer/updater.
  4. Profession enchants are not recognised. this is a known bug in bbTips.
